### Requirements
`Build`
- C++20 compiler
- cmake 3.11
### Build
```
- At the root of the repository in a terminal
mkdir build
cd build
cmake ..
cmake --build .
```
### Usage
```
USAGE:
    ./2048
HOTKEYS:
    Keyboard (american):
        UP : Z
        DOWN : Q
        LEFT : S
        RIGHT : D
    Joystick
```
